# Currículo Civil UFSC

Este projeto contém tudo o que é necessário para rodar o aplicativo localmente.

## Como Executar Localmente

**Pré-requisitos:** Node.js

1. Instalar as dependências:
   `npm install`
2. Executar o aplicativo:
   `npm run dev`
